### 其他各种语言
还支持其他各种语言：

![Postman把请求生成其他各种语言的代码](../assets/img/post_generat_other_lang_code.png)

目前支持的语言有：
* HTTP
* C (LibCurl)
* cURL
* C#(RestSharp)
* Go
* Java
  * OK HTTP
  * Unirest
* Javascript
* NodeJS
* Objective-C(NSURL)
* OCaml(Cohttp)
* PHP
* Python
* Ruby(NET::Http)
* Shell
* Swift(NSURL)

代码生成工具的好处是：在写调用此API的代码时，就可以参考对应代码，甚至拷贝粘贴对应代码，即可。
